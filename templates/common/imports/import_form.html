Pasos a seguir para la importación

Sube un archivo .csv para crear varios registros de una sola vez. Sigue estos pasos para que todo salga bien.

Descarga la plantilla

Baja la plantilla (Excel) con las columnas correctas y úsala como base.

Edita el archivo con tus datos

Completa una fila por cada registro, respetando el orden de las columnas.

Guárdalo en formato CSV UTF-8

En Excel: «Guardar como» → CSV UTF-8, así se conservan tildes y ñ.

Selecciona o arrastra el archivo en el recuadro

Suéltalo en la zona de carga o usa «Seleccionar archivo».

Pulsa importar

Confirma y espera el resumen: verás los registros creados o los errores por fila.
