# Carpeta JS - Librerías Externas

Aquí se deben colocar todos los archivos JavaScript de librerías externas como:

- jQuery
- Bootstrap JS
- DataTables JS y extensiones
- Otros frameworks/librerías de terceros

## Ejemplo de archivos:

- jquery-3.7.0.min.js
- bootstrap.min.js
- jquery.dataTables.min.js
- dataTables.responsive.min.js
- dataTables.buttons.min.js
- buttons.html5.min.js
- jszip.min.js
- pdfmake.min.js
