# Carpeta JS - Scripts Personalizados

Aquí se deben colocar todos los archivos JavaScript personalizados del proyecto:

- Scripts globales del proyecto
- Scripts específicos por app
- Funciones utilitarias
- Configuraciones de librerías

## Ejemplo de archivos:

- main.js (script principal global)
- clients.js (funciones específicas de clients)
- services.js (funciones específicas de services)
- utils.js (funciones utilitarias)
- datatables-config.js (configuración de DataTables)
