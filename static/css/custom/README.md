# Carpeta CSS - Estilos Personalizados

Aquí se deben colocar todos los archivos CSS personalizados del proyecto:

- Estilos globales del proyecto
- Estilos específicos por app
- Temas personalizados
- Sobrescritura de estilos de librerías

## Ejemplo de archivos:

- main.css (estilos globales)
- clients.css (estilos específicos de clients)
- services.css (estilos específicos de services)
- theme.css (tema personalizado)
- overrides.css (sobrescribir estilos de librerías)
