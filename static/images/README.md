# Carpeta de Imágenes

Aquí se deben colocar todas las imágenes estáticas del proyecto:

- Logos del proyecto
- Iconos personalizados
- Imágenes de fondo
- Placeholders
- Sprites de iconos

## Estructura sugerida:

- logos/ (logotipos del proyecto)
- icons/ (iconos personalizados)
- backgrounds/ (imágenes de fondo)
- ui/ (elementos de interfaz)
