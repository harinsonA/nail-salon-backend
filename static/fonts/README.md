# Carpeta de Fuentes

Aquí se deben colocar todos los archivos de fuentes del proyecto:

- Fuentes de iconos (FontAwesome, Material Icons, etc.)
- Fuentes tipográficas personalizadas
- Archivos WOFF, WOFF2, TTF, EOT

## Ejemplo de archivos:

- fontawesome-webfont.woff2
- custom-font.ttf
- icons.woff
